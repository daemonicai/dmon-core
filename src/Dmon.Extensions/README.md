# Dmon.Extensions

Extension SDK surface for building dmon extensions.

Provides helpers and base infrastructure for implementing `IDmonExtension` packages:
- Middleware pipeline integration
- Per-extension configuration support
- Utilities for registering `AIFunction` tools with dmon

Depends on `Dmon.Protocol` and `Microsoft.Extensions.AI`. Reference this package when building a NuGet-packaged dmon extension.
